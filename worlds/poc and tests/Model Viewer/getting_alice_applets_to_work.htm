How to play Alice worlds in your web browser:

NOTE!  You may need to have administrator privileges to download this software.  If you are using a computer in a public lab, you may not be able to download these, and you will not be able to see the Alice world without them.    We're sorry about that, but this is not something the Alice system can fix you will need to talk with your local system administrator.

You must download and install the following in order to view Alice 2.x worlds on the web:

  1. Java
  2. Java 3D
  3. Java Media Framework

The following are instructions on how to install these tools. Note that these instructions are for Internet Explorer users.

Installing Java

If you already have Java, you can skip to Installing Java 3D. If you are not sure whether you have Java, you can check by clicking here and clicking Verify Installation.

  

Step 1: Go to the Java download page.

  

Step 2: Click the Free Java Download link.

Step 3: Click the Begin Download link.

If you receive an error message, look for the yellow Internet Explorer bar at the top of your browser window.

Click on the bar and select "Install ActiveX Control?"

Step 4: Click "Install" from the popup.

Select Typical Installation" from the installation screen and follow the on-screen installation instructions.

You might need to restart your machine after Java is installed.

 

Installing Java 3D

  

Step 1: Go to the Java 3D download page.

 

Step 2: Click the download link under Download Java 3D 1.x Software

Step 3: Select your platform from the dropdown box

 

Tick the checkbox labelled "I agree to the Software License Agreement"

 

Then click Continue >>

Step 4: Click the Required download file link

Step 5: Click "Run" from the popup.

 

 

 

 

 

 

 

Click on "Run" again. Follow the on-screen installation instructions.

 

Installing Java Media Framework

  

Step 1: Go to the JMF download page.

 

Step 2: Click the download link under Download JMF 2.1.1x Software.

Step 3: Select your platform from the dropdown box

 

Tick the checkbox labelled "I agree to the Software License Agreement"

 

Then click Continue >>

Step 4: Click the Windows Performance Pack download link.

Step 5: Click "Run" from the popup.

 

 

 

 

 

 

 

Click on "Run" again. Follow the on-screen installation instructions.

 

You might need to restart your machine after JMF is installed.

 

Final Installation Instructions

  

Step 1: After you have finished installing Java, Java 3D and Java Media Framework, open your Alice world saved out as a web page (it should have a .html extension).

If your world does not load, look for the yellow Internet Expolorer bar at the top of your browser window.

Click the bar and select Allow Blocked Content.

 

 

 

 

Voila! Your Alice world in a webpage.

 

If you still have problems viewing your world, first try restarting your computer. Then submit a bug report or post a message on our forums.
